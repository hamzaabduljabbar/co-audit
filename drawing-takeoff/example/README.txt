EXAMPLE OUTPUT
==============

These two files show what the skill produces when run on a real 89-page
architectural drawing set. They are here so you can see the output before you
run it on your own drawings.

  drawing.md          The one-page set index the skill auto-generates. Read it
                      first on any question - it says which sheet holds what, so
                      you query straight to the right place. (Source path/name
                      genericized to SET.pdf.)

  example-output.txt  Real console output: object counts, a sample of the door
                      list with sizes/materials, and the sanity check. Room names
                      were removed - the sizes and materials shown are the real
                      extracted values.

You do NOT get a pre-built database for your drawings in this zip - you generate
your own by running `build` + `schedules` on your PDF (see SETUP.md). The whole
point is that it works on YOUR set, not a demo.
